# General
An application for tracking students' attendance

## Technologies
Project is created with:
* Java 17.0.6
* Maven 3.9.3
* Javafx 19.0.2.1
	
## Setup
To run this project:

```
$ cd ../project_directory
$ mvnw clean javafx:run
```

To build and run the project:

```
$ cd ../project_directory
$ mvnw clean package
$ cd project_directory/target
$ ./attenadance-version-shaded.jar
```
